## 👩‍🏫 Como rodar localmente

- `git clone https://github.com/Gustavoohrq/ezequiel-pulpfiction.git`;
- `cd ezequiel-pulpfiction`;
- `npm install`;
- `yarn start`;
- Acessar `localhost:3000`;
